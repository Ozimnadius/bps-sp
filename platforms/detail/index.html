СмартВиста Интеграционная платформа (SVIP)

Платформа предназначена для реализации интеграции большого количества систем путем настройки гибкой маршрутизации сообщений с использованием оркестратора бизнес-процессов и для преобразования форматов данных.

Платформа используется и для добавления новых сервисов,
расширяющих бизнес функциональность внешних систем.

решаемые задачи

Платежный шлюз
Транзакционный сервис обработки данных
Предоставление API интерфейсов
Универсальная ESB шина и преобразователь протоколов
Платежный шлюз

Функциональность

  • Менеджер бизнес-процессов для управления различными сервисами и каналами
  • Облегченная ESB, обеспечивающая интеграцию без кодирования
  • Редакторы шаблонов сообщений таких как ISO 8583, XML, JSON, OAS (swagger) и т.д.
  • Настройки соответствия полей различных сообщений
  • Поддержка большого числа коннекторов к различным системам
  • Большое число сервисов доступных из коробки для настройки логики процессинга сообщений
